Sistema Dindin

API RESTful desenvolvida para gerenciar transações financeiras pessoais. Através dela, os usuários podem cadastrar e gerenciar suas transações, categorizá-las e obter um resumo financeiro. Cada usuário tem acesso exclusivo às suas informações, garantindo a segurança e privacidade dos dados.

Sistema PDV

Desenvolvida em grupo, é uma REST API para PDV (Ponto de Venda). Sendo um sistema de gerenciamento para frente de caixa que permite a criação e manipulação de dados relacionados a categorias, clientes, pedidos, produtos e usuários. A API possibilita operações de cadastro, atualização e listagem de informações essenciais para a gestão eficiente de um PDV, utilizando um banco de dados PostgreSQL, onde todas as informações são armazenadas com segurança.


Link: GitHub | Ponto de Venda
